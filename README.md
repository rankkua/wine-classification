# Wine Classification Project

Classify wines into their respective classes using machine learning. 
This project includes exploratory data analysis (EDA), model training and evaluation, 
and visualizations of model performance and feature importance.

## Project Goals
* Perform exploratory data analysis (EDA) to understand wine features.
* Classify wines based on chemical properties using Random Forest.
* Compare results to another model.
* Evaluate model performance with accuracy, confusion matrix, etc.
* Visualize results, feature importances, and model performance.
